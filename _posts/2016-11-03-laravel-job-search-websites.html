Job Search / Accelar Job/Internship search website using Laravel 5.3 and Angular JS. This website has three type of Users, Students, Company and Admin. Student can create profile, add their details and Internship choice. Company can add their requirements and internships options. Admin can see there internships and send emails to students and companies.

Admin dashboard built using Angular JS and UI Router. User and Front facing pages create using Bootstrap and use Laravel blade template.

Bootstrap, Slick Carousels, Parsley JS used for smooth User experience.

MariaDB used as Database

We use LAMP on Linux and XAMP on Windows

Atom, Sublime, VSCode and PHPStorm used as Code Editors and IDE

We took 1 Month to complete this Project.
