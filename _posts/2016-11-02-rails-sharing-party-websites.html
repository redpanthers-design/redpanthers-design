Sharing Party

Sharing Party is an Italian website/social media website to share details of different parties, events and halls. It is a Rails 4.3 website and hosted on Heroku.

SHaring party consist 2 type of Users, Normal and Admin User. Admin User manage entire website, where Normal users can only add Parties, Follow other users.

One of the interesting feature is, we add user recognition javascript library to cross check profile images. None of the Frontend framework used on it. jQuery, Turbolinks, Owl Carousel, Google map are used in it.

Bootstrap used as CSS framework. Zurb Ink used for Email template.
